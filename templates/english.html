Projects

Productivity App

Task management system with MySQL, the system allows users to register, query, update and delete tasks associated with your account. Each task includes information such as title, description, status, and creation time, with relationships between users and tasks to ensure integrity. SQL queries allow you to filter tasks by user, view pending or completed tasks, and manage tasks. information efficiently.

Junior Shop

This project consists of creating a complete website for a pet store called Junior Shop. The objective is to transform a paper prototype into a functional web page, using the best web development practices with HTML and CSS. The end result is an attractive website. Users can easily navigate through the product categories, the site is fast, modern and SEO optimized, with a fluid experience on any device.

Memory Game

This project is an interactive memory game created using JavaScript., HTML and CSS, as part of a course on the fundamentals of the JavaScript programming language. The game is based on the images of the characters from the famous television series The Simpsons and consumes a phrase API from the series to generate dynamic content in the game. jQuery was implemented to simplify DOM manipulation.
